WELCOME TO ROADS!

A one-time system setup for your cloud service




This is the sample Python project deployed by Roads. It is pre-configured to serve webpages over the internet and you can find the original code on github. For now this is now just a bare bones Django project, which you can use as a springboard for your own project! For more information about Django, you can checkout the getting started guide on djangoproject.com.

POSTGRESQL

The world's most advanced open-source relational database



Your Roads came set up with an advanced SQL database, and your Python project has been configured to work with it. The pure SQL command line interface can be reached by SSH'ing into your machine, running `su postgres` and then running `psql`. For more information about it, you can try the getting started guide..
